Compound Assignment With Augmented Subtraction

Like the += operator, -= subtracts a number from a variable muyVar = myVar - 5; will subtract 5 from myVar. This can be rewritten as: myVar -= 5;

let a = 11; let b = 9; let c = 3; a -= 6; b -= 15; c -= 1; console.log(a); console.log(b); console.log(c);

Compound Assignment With Augmented Multiplication

let a = 5; let b = 12; let c = 4.6; a *= 5; b *= 3; c *= 10; console.log(a); console.log(b); console.log(c);
